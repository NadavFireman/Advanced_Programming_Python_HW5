# Advanced Programming: Machine Learning Fundamentals (HW5)

This repository contains the solution for Home Assignment 5 in the Advanced Programming course (Shenkar College, IE&M).

The assignment focuses on applying data science principles to build and evaluate predictive models, utilizing the provided dataset on diabetes symptoms.

## Key Skills Demonstrated

* **Data Exploration and Feature Engineering:** Loading and preparing data from the **CSV file (`diabetes_sympthoms.csv`)**, including handling categorical and binary features.
* **Model Selection and Training:** Implementation of a core **Machine Learning model** (e.g., Logistic Regression, Decision Tree, etc.) for classification.
* **Model Evaluation:** Assessing model performance using standard metrics (e.g., Accuracy, Confusion Matrix, AUC).
* **Statistical Insights:** Deriving conclusions about the correlation between reported symptoms and the classification outcome.

## Repository Content

* **`Advanced_Programming_HW5.ipynb`**: Full Python solution notebook with code and analysis results.
* **`diabetes_sympthoms.csv`**: The raw dataset used for training and prediction.
* **`Assignment_Instructions_5.pdf`**: Original assignment instructions.
